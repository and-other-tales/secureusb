SecureUSB Installation Complete!

⚠️ Important: SecureUSB requires initial configuration before it can protect your USB ports.

Required: Initial Setup

  1. Open Terminal (Applications → Utilities → Terminal)
  2. Run the setup wizard:
    sudo secureusb-macos setup
  3. Install Google Authenticator on your phone (or any TOTP app)
  4. Scan the QR code displayed in Terminal
  5. Save the recovery codes shown
  6. Enter a TOTP code to verify the setup

Managing SecureUSB

Use these commands in Terminal:

  • secureusb-macos status - Check if daemon is running
  • secureusb-macos start - Start USB protection
  • secureusb-macos stop - Stop USB protection
  • secureusb-macos restart - Restart the daemon
  • secureusb-macos setup - Run setup wizard again

How It Works

  1. When you plug in a USB device, a dialog will appear
  2. Open Google Authenticator on your phone
  3. Enter the 6-digit code shown for SecureUSB
  4. Choose to Connect, allow Power Only, or Deny the device

Configuration Files

  • Config: /Library/Application Support/SecureUSB/
  • Logs: /Library/Logs/SecureUSB/
  • Installation: /opt/secureusb/
